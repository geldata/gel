======
Fly.io
======

:edb-alt-title: Deploying applications built on Gel Cloud to Fly.io

Gel Cloud is shutting down and Fly.io deployment guides for the managed
service are retired. Export your legacy hosted data with :gelcmd:`dump` and
deploy a self-managed Gel instance to Fly.io using the instructions in
:ref:`ref_guide_deployment_fly_io`.
