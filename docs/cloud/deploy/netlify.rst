=======
Netlify
=======

:edb-alt-title: Deploying applications built on Gel Cloud to Netlify

Gel Cloud is shutting down and Netlify deployments against the service are no
longer supported. Dump your hosted data with :gelcmd:`dump` and connect your
Netlify app to a self-managed Gel instance following the guidance in
:ref:`ref_guide_deployment`.
