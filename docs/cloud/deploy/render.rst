======
Render
======

:edb-alt-title: Deploying applications built on Gel Cloud to Render

Gel Cloud is shutting down and Render deployment instructions for the managed
service have been retired. Export your data with :gelcmd:`dump` and deploy a
self-hosted Gel instance using the Render guidance in
:ref:`ref_guide_deployment`.
