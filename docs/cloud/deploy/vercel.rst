======
Vercel
======

:edb-alt-title: Deploying applications built on Gel Cloud to Vercel

Gel Cloud is shutting down and no longer supports deploying to Vercel. Export
your data from the legacy service using :gelcmd:`dump`, then follow the Vercel
steps in :ref:`ref_guide_deployment` to connect your app to a self-managed Gel
instance.
